Connect Supabase

Enter your Supabase credentials to launch your blog.

1

Create a free project at supabase.com

New Project → choose a name and region

2

Run the SQL schema

Supabase → SQL Editor → paste schema.sql → Run

3

Create storage bucket

Storage → New bucket → name: blog-images → Public: ON

4

Get your API credentials

Settings → API → copy Project URL and anon public key

✅ Checklist:
• URL format: https://xxx.supabase.co (no trailing slash)
• Use anon public key (starts with eyJ...)
• Run schema.sql first in Supabase SQL Editor
• Create blog-images bucket with Public ON
